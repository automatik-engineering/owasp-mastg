A sample application demonstrating the @MASTG-TOOL-0147 library. The app showcases detection capabilities for root, emulators, and debuggers. The library implements checks using native code written in Rust to make bypassing more difficult. Provides a test interface to evaluate various RASP detection techniques including root indicators, emulator characteristics, and debugger attachment.
