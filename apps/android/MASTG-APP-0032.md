A sample application demonstrating the @MASTG-TOOL-0146 library, a root checking library for Android. The app provides a simple interface to test various root detection techniques including checking for root management apps, dangerous apps, root cloaking apps, test keys, dangerous properties, BusyBox and `su` binaries, and read-write system partitions.
